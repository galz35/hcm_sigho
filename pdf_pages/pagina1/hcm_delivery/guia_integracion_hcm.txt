================================================================================
  PASO 1: ABRIR EL EDITOR DEL SITIO DE CARRERAS
================================================================================

  1. Inicia sesión en Oracle Cloud HCM
  2. Navega a: Setup and Maintenance → Recruiting → Career Sites
  3. Busca tu sitio "prueba 1" (CX_1001)
  4. Haz clic en "Edit" (Editar)

================================================================================
  PASO 2: PEGAR EL CSS EN EL TEMA
================================================================================

  1. En el panel izquierdo → pestaña "Theme" (Tema)
  2. Busca la sección "Custom CSS"
  3. BORRA todo lo que haya ahí
  4. Abre el archivo hcm_styles.css en tu editor de texto
  5. Selecciona TODO (Ctrl+A) → Copiar (Ctrl+C)
  6. Pega (Ctrl+V) en la caja "Custom CSS" de Oracle
  7. Haz clic en "Save"

  ¿QUÉ INCLUYE ESTE CSS?
    ✓ Tailwind CSS v2 importado (@import)
    ✓ Carrusel automático del Hero (4 fotos, rotación cada 5s)
    ✓ Degradado blanco sobre imagen del Hero
    ✓ Estilos del buscador nativo de Oracle (forma pill, botón rojo)
    ✓ Estilos del menú nativo de Oracle (Montserrat, colores Claro)
    ✓ Botón de perfil estilizado (rojo, redondeado)
    ✓ Ocultamiento del botón "Soy empleado"
    ✓ Todas las clases utilitarias del diseño

================================================================================
  PASO 3: PEGAR EL HTML EN LA PÁGINA DE INICIO
================================================================================

  1. En el panel izquierdo → pestaña "Pages" (Páginas)
  2. Selecciona la página Home (Splash Page)
  3. Si ya tienes un bloque "Custom HTML":
     → Haz clic en el icono de editar (lápiz)
     Si no lo tienes:
     → Agrega una fila de 1 columna a ancho completo
     → Arrastra el componente "HTML" a esa fila
     → Haz clic en el lápiz para editar
  4. BORRA todo lo que haya en el editor de código
  5. Abre el archivo hcm_template.html
  6. Selecciona TODO (Ctrl+A) → Copiar (Ctrl+C)
  7. Pega (Ctrl+V) en el editor de código de Oracle
  8. Haz clic en "Done" (Listo)
  9. Haz clic en "Save"

  NOTAS:
    - Este HTML NO tiene <script>, <style>, <html>, <head> ni <body>
    - NO tiene header/menú (Oracle pone el suyo propio)
    - NO tiene buscador decorativo (Oracle pone el suyo funcional)
    - NO tiene botones flotantes (Oracle maneja chat/perfil)
    - NO debería darte ningún error de "elementos restringidos"
