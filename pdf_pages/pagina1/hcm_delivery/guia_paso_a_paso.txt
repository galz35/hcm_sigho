PASO 1: CONFIGURAR EL CSS GLOBAL (Solo una vez)
================================================================================

1.1) Abre tu Career Site en Oracle HCM
     Navegacion: Herramientas > Configuracion del Career Site

1.2) Ve a la pestana "Theme" (Tema)

1.3) Busca la seccion "Custom CSS" (CSS Personalizado)

1.4) BORRA todo el CSS que tengas ahi actualmente

1.5) Abre el archivo: hcm_styles.css
     Selecciona TODO el contenido (Ctrl+A), copia (Ctrl+C)

1.6) Pega (Ctrl+V) en el campo "Custom CSS" de Oracle

1.7) Haz clic en "Save" (Guardar)

     NOTA: Este archivo CSS contiene los estilos de TODAS las paginas
     (Homepage, Cultura, Beneficios, Proceso y Contacto).
     Solo necesitas pegarlo UNA VEZ.

================================================================================
NOTAS IMPORTANTES
================================================================================

* Si Oracle te da el error "codigo personalizado incluye elementos
  restringidos", verifica que NO hayas modificado los archivos HTML
  despues de copiarlos. Los archivos fueron validados automaticamente
  y no contienen ninguna etiqueta bloqueada.

* Si necesitas editar el HTML despues, NUNCA uses estas etiquetas:
  <html>, <head>, <body>, <style>, <script>, <form>
  Tampoco uses el atributo style="..." ni enlaces que terminen en .html

* Las fuentes de Google y Tailwind CSS se cargan automaticamente
  mediante etiquetas <link> al inicio de cada bloque HTML.

* El CSS es GLOBAL: un solo archivo para todas las paginas.
  Si cambias colores o fuentes, solo editas hcm_styles.css
  y lo vuelves a pegar en Theme > Custom CSS.
